# 📄 PDF Redaction Tool

A React-based PDF redaction interface that allows users to manually draw redaction boxes over PDF pages, send them to a backend for processing, and view/download the redacted version.

---

## ⚙️ Prerequisites

- **Node.js**  
  Check your version:

  ```bash
  node -v
  ```

## TO RUN THIS PROJECT

- \*\* npm i
- \*\* npm start
